Help

This page helps you understand how the program works and how to contact support.

Pulls

Currency: This is the currency that you currently have.

Actual Amount: Enter the amount of currency you have. For example, if you have 120 diamonds, it will calculate how many pulls that equals.

Actual Pulls: Displays how many pulls you currently have for each currency.

Needed Pulls: Shows how many pulls you need in total.

Total Pulls: The total amount of pulls you have made.

Needed Pull: The remaining pulls needed to reach your goal.

Outfit Items: The number of clothing items you still need.

Make Pulls

You can select the number of pulls and the currency you want to use. When clicking "Make Pulls", the information will be added to the history, including the amount and selected stars.

Buttons

Navigation buttons in the application:

  • History: View your manual pull history and pity count.
  • Close: Closes the application.
  • Help: Opens this help page.
  • Reset: Resets all values.


History

View your pull history, including the date, number of pulls, star rating, and pity count.
